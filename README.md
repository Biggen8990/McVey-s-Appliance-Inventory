Appliance Inventory Management App
 
A feature-rich, console-based inventory app for multi-store appliance businesses.
 Tracks, edits, archives, restores, and reports on appliances with built-in audit logging, backup/restore, notes, advanced search, and robust validation.
 Includes easy CSV/JSON import and export, bulk actions, and a business-friendly archive system.
 
Designed to be easily maintained, audit-ready, and prepared for future web app migration.
